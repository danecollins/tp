Today I'm releasing the first beta release of Track Places. For those of you that saw the preview you'll notice these changes:

  1. You can now create your own account.
  2. There is now a Guest User Mode so people can browse without creating an account.
  3. A Cuisine field has been added to places.
  4. You can now search for places by name and add them to your list if they are not already there.
  5. There is now an About page that gives some background info.

I hope you like the changes. I'm sure you'll run into issues and as you do please add them as comments to this post or email them to dane@dacxl.com

Thanks for trying out Track Places

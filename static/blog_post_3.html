You'll see the following changes:

New place detail view
The place detail view is now a table intead of a list and it includes images for the outdoor, dog friendly and rating fields. You'll still set these as text when you edit them.
Rating field range
Because of the new images for ratings it's now important that you use -1 for a place you don't want to go back to, 0 for a place you have not rated and 1, 2 or 3 to rate a place you have been to. These are the only values that will display as images.
Cities in Search are now links to details
In v0.01 you could search for places on the search page but your only option then was to add them. Now the cities are links to the details page so you can look at them before deciding whether to add them.
iPhone Add to Home Screen now adds an icon
If you use the Add to Home Screen function on the iPhone you'll now get a TP icon image

I hope you like the changes. I'm sure you'll run into issues and as you do please add them as comments to this post or email them to dane@dacxl.com
